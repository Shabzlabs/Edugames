Contact ShabzLabs

Thank you for visiting ShabzLabs. This website is designed to support primary students through interactive learning games and simple educational resources.

General Questions

If you have questions about the website, the games, or how the activities can be used for learning, you can contact ShabzLabs by email.

Teacher Feedback

Feedback from teachers, parents, and educators is helpful as ShabzLabs continues to grow. Suggestions about game design, classroom usefulness, accessibility, or curriculum connections are welcome.

Game Suggestions

If there is a topic you would like to see added, such as math, language, science, or social studies games for primary grades, you can send a suggestion.

Technical Issues

If a game link does not work, a page does not load correctly, or something seems broken, please include the name of the game or page when reaching out.
